C++简介

概述

C++ 是一种静态类型的、编译式的、通用的、大小写敏感的、不规则的编程语言，支持过程化编程、面向对象编程和泛型编程。

组成

  • 核心语言（变量、数据类型、常量）
  • C++标准库（大量函数，操作文件、字符串）
  • STL（操作数据结构）

面向对象开发四大特性

  • 封装
  • 抽象
  • 集成
  • 多态

数据类型

基本的内置类型


typedef声明

您可以使用 typedef 为一个已有的类型取一个新的名字。

typedef double newname

枚举类型

enum 枚举名{red, green = 5, blue} rgb，默认red = 1, green = 5, blue = 6。

数组

所有的数组都是由连续的内存位置组成。最低的地址对应第一个元素，最高的地址对应最后一个元素。

声明

double example[10];

初始化

double balance[5] = {1000.0, 2.0, 3.4, 7.0, 50.0};

double balance[] = {1000.0, 2.0, 3.4, 7.0, 50.0};

传参

  • 您可以通过指定不带索引的数组名称来生成一个指向数组中第一个元素的指针。
  • 您可以通过指定不带索引的数组名称来给函数传递一个指向数组的指针。
  • C++ 允许从函数返回数组。？是不是也是返回数组名即可

字符串

C风格

char greeting[6] = {'H', 'e', 'l', 'l', 'o', '\0'};

char greeting[] = "Hello";

其实，您不需要把 null 字符放在字符串常量的末尾。C++ 编译器会在初始化数组时，自动把 ‘\0’ 放在字符串的末尾。

C++风格

C++ 标准库提供了 string 类类型，支持上述所有的操作，另外还增加了其他更多的功能。

注意头文件的不同

指针

  • 空指针：表示不明确指向内存，但指向特定的内存位置。
  • 指针和数组的区别：
    • 作为形参，指针和数组一样。
    • 作为返回值，指针和数组一样。
    • 不同的是，数组名表示数组的首地址，可以++、--、或者取值运算*获得地址里的值。这和指向数组的指针一样，但是不能讲数组名作为左值来修改它的地址。
  • 指针数组。
  • 指向指针的指针。

引用

  • 引用和指针的区别：
    • 不存在空引用，引用必须初始化。
    • 引用初始化后，不可更改。
  • 引用作为参数。
  • 引用作为返回值。

变量

定义

int a; int b = 0;//a、b分配了内存，b获得了初值。

声明

extern int a;//声明a是一个已经定义了的外部变量，甚至a可以是不同文件中定义的。

注意这个外部变量和本地的局部变量的划分。

左值和右值

左值：指向内存位置的表达式成为左值，可以在赋值号的左边或右边。
右值：指的是存储在内存中的具体数值，是不能被赋值的表达式，只能赋值别人。

作用域

  • 局部变量：在函数或者代码块内声明的变量。
  • 形参：函数参数定义中声明的变量。
  • 全局变量：函数外部声明的变量。

全局变量在整个程序周期都是有效的，可以被任意函数访问。注意全局和局部重名情况的处理。

全局变量会默认初始化，局部变量不会。

  • signed

  • long
  • short

类型限定符

常量

常量是不会被改变的量，可以是任何的基础类型。

整数常量

可以是十进制、八进制、十六进制。

整数常量也可以带一个后缀，后缀是 U 和 L 的组合，U 表示无符号整数（unsigned），L 表示长整数（long）。后缀可以是大写，也可以是小写，U 和 L 的顺序任意。

浮点常量

有小数形式和指数形式。

布尔常量

略。

字符常量

字符常量是括在单引号中。如果常量以 L（仅当大写时）开头，则表示它是一个宽字符常量（例如 L’x’），此时它必须存储在 wchar_t 类型的变量中。否则，它就是一个窄字符常量（例如 ‘x’），此时它可以存储在 char 类型的简单变量中。

字符常量可以是一个普通的字符（例如 ‘x’）、一个转义序列（例如 ‘\t’），或一个通用的字符（例如 ‘\u02C0’）。

在 C++ 中，有一些特定的字符，当它们前面有反斜杠时，它们就具有特殊的含义，被用来表示如换行符（\n）或制表符（\t）等。下表列出了一些这样的转义序列码：

存储类

auto

自 C++ 11 以来，auto 关键字用于两种情况：声明变量时根据初始化表达式自动推断该变量的类型、声明函数时函数返回值的占位符。

static

  • 局部变量：在程序的生命周期内保持局部变量的存在，而不需要在每次它进入和离开作用域时进行创建和销毁。因此，使用 static 修饰局部变量可以在函数调用之间保持局部变量的值。
  • 全局变量：当 static 修饰全局变量时，会使变量的作用域限制在声明它的文件内。
  • 类成员变量：会导致仅有一个该成员的副本被类的所有对象共享。

extern

当您有多个文件且定义了一个可以在其他文件中使用的全局变量或函数时，可以在其他文件中使用 extern 来得到已定义的变量或函数的引用。可以这么理解，extern 是用来在另一个文件中声明一个全局变量或函数。
